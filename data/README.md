# Dataset Information

This folder contains information about the dataset used in the Silent Burnout AI project.

The project uses the Open University Learning Analytics Dataset (OULAD), which contains student demographic, assessment, and LMS interaction data.

Due to GitHub file size limits, large dataset files are not uploaded directly to this repository.

## Required Dataset Files

- studentInfo.csv
- studentVle.csv
- studentAssessment.csv
