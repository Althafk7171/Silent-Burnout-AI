# Project Reports

This folder contains academic documentation and presentation files for the Silent Burnout AI project.

## Files Included

- `Research_Paper_Final.pdf` - Research paper based on the project methodology and results
- `silentburnout.pdf` - Final capstone project report
- `Final-PPT.pptx` - Final project presentation

## Description

These files include the project background, problem statement, literature review, system design, methodology, implementation, results, testing, conclusion, and future scope.

The documents are included for academic submission, reference, and project documentation purposes.
